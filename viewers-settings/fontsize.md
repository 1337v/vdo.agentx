# \&fontsize

## Options

| Value           | Description                     |
| --------------- | ------------------------------- |
| (integer value) | font size value as a percentage |

## Details

Lets you set font-size of the guest labels or closed captions.



#### Related parameters:

* `&cc`
* `&showlabels`
