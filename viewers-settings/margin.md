# \&margin

## Options

| Value            | Description        |
| ---------------- | ------------------ |
| (integer value)  | margin value in px |
| (no value given) | 10 px              |

## Details

Adds px around the videos for some spacing.

Added as a toggle to the director's room and it can be customized.

Defaults to 10px if no value is given.
