# \&margin

## Options

| Value            | Description         |
| ---------------- | ------------------- |
| (integer value)  | margin value in px  |
| (no value given) | margin value = 10px |

## Details

Adds 10px around the videos for some spacing, by default. It can be customized.

It's also available as a toggle to the director's room.
