# \&screensharefps

## Aliases

* `&ssfps`

## Options

| Value           | Description                        |
| --------------- | ---------------------------------- |
| (integer value) | will target this framerate. Eg: 60 |

## Details

Set a target FPS for your screenshare, when you screenshare as a second screen in a room. Use \&framerate
