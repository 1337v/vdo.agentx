# \&aec

## Aliases

* `&ec`

## Options

| Value | Description |
| ----- | ----------- |
| 0     | Off         |
| 1     | On          |

## Details

May need to be disabled to use [`&stereo`](../general-settings/stereo.md) on some older browsers
