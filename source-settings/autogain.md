# \&autogain

Sender-Side Option! ([`&push`](push.md))

## Aliases

* `&ag`

## Options

| Value | Description |
| ----- | ----------- |
| 0     | Off         |
| 1     | On          |

## Details

Adding `&autogain` to a source link, sets whether audio auto-normalization is ON or OFF. Your browser will try to keep optimum audio levels.
