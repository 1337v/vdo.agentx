# \&nopreview

## Aliases

* `&np`

## Details

If you don't want to see yourself when in a group chat, this will hide your video.

{% hint style="danger" %}
**iOS users:** This flag will break stuff; You must have a preview if using those devices and wanting to publish.
{% endhint %}
