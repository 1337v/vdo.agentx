# \&limiter

## Details

A limiter is like an aggressive compressor.\
\
The limiter is **off by default**.

{% hint style="warning" %}
Enables the audio processing pipeline.
{% endhint %}

{% hint style="danger" %}
It cannot be used along with the [`&compressor`](and-compressor.md) option.
{% endhint %}
