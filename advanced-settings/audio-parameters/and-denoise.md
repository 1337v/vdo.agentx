# \&denoise

## Aliases

* `&dn`

## Options

| Value | Description |
| ----- | ----------- |
| 0     | Filter Off  |
| 1     | Filter On   |

## Details

Adding `&denoise` to a source link turns audio noise reduction filter ON or OFF.
