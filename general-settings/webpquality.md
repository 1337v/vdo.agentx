# \&webpquality

## Aliases

* `&webpq`
* `&wq`

## Options

| Value | Description            |
| ----- | ---------------------- |
| 5     | 270p @ 15fps (default) |
| 4     | 270p @ 35fps           |
| 3     | 270p @ 50fps           |
| 2     | 360p                   |
| 1     | 720p                   |
| 0     | 1080p                  |

## Details

You add this parameter to the director (or designated broadcaster) and it then sets the the quality target for the [`&webp`](../advanced-settings.md#webp) mode.
